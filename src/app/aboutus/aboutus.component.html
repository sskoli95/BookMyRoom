About BookMyRoom

A project created by Sadanand Koli and Pratik Nikam

Project created as part of cdac curriculum The Book My Room is Hotel booking web application which is created with the purpose of providing complete solutions for Hotel owners as well as customers through a single get- way using the internet. This system allows the customer's to book Hotel Rooms according to their requirements with affordable price by using internet. Hotel owners will be able to upload room details and specifications and the pricing.
